# 更新日志

本文档用于记录 Bilibili-Boost 各版本的主要变更。

## v1.3.1

### 新增

- 新增 Firefox 本地安装兼容适配

### 文档

- 更新 README 中的 Firefox 本地安装与兼容说明

## v1.3.0

### 新增

- 新增已看记录 `导出` / `导入` 功能，可通过控制面板管理本地记录
- 导出文件使用 `JSON` 格式，仅包含插件自己的已看记录数据

### 优化

- 手动标记 `已看完` 后再次取消时，会优先恢复到标记前的观看进度
- 控制面板中的当前视频操作支持 `取消已看完`

## v1.2.0

### 新增

- 新增 `标记看过` 功能，支持自动记录视频观看进度
- 新增 `已看 XX%` / `已看完` 两种观看状态展示
- 新增视频详情页状态区域，可手动执行 `标记已看完` 和 `清除记录`
- 新增缩略图右上角已看状态徽标
- 新增 UP 主空间主页视频卡片底部快捷标记按钮
- 新增插件控制面板，可统一管理 `标记看过` 和 `合集增强` 开关

### 存储与同步

- 已看记录改为由插件本地持久化保存
- 设置项使用 `chrome.storage.local`
- 观看记录使用 `IndexedDB`
- 新增跨标签页的已看记录同步更新

### 保留

- 完整保留 `v1.1.0` 的 `合集增强` 能力

## v1.1.0

### 新增

- 首次上线 `合集增强` 功能
- 在视频页右侧播放列表区域新增 `展开标题` 按钮
- 在视频页右侧播放列表区域新增 `展开列表` 按钮
- 支持展开被省略号截断的视频标题
- 支持拉高播放列表区域，便于浏览分 P 与合集内容

### 样式

- 按钮样式贴近 B 站原生界面风格
